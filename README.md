# MultiagentSystems_ANAC
This repository contains an agent compatible for the ANAC competition 2023 (ANL 2023) Rules. This agent won the internal class competition of Technical University of Crete. The main goal is to succeed both in Utility and Social Welfare. The agent follows these methods:
* Bidding Policy: time-depended threshold
* Fictitious Play: We keep our opponents preferences to suggest something that both accept.
* Opponent modeling: We categorize the opponents to greedy or explorative.
